# Seed Audit

This file stores per-run seed-family verification state. It is operational state, not the canonical business ledger.

## Audit Rules

- Every seed family must appear once.
- `结果去向` must be one of: `Main Pool`, `Tracking List`, or an explicit unresolved status.
- Audit counts and destination counts must match the actual section contents before export.
- This file can be recreated for each run. Do not use it as a long-term business ledger.
